## 📝 Customization Guide

### 1. Update Personal Information

Edit `index.html` to customize your information:

- **Profile Image**: Replace `assets/profile.jpg` with your photo
- **Resume**: Replace `assets/resume.pdf` with your resume
- **Contact Details**: Update email, phone, and location
- **Social Links**: Update LinkedIn and GitHub URLs

### 2. Modify Content

Update these sections in `index.html`:

- **About Section**: Describe your background and expertise
- **Skills Section**: Add/remove technical skills
- **Experience Section**: Add your work history
- **Projects Section**: Showcase your key projects

### 3. Customize Styling

Edit `style.css` CSS variables to match your brand:

```css
:root {
    --primary-color: #3b82f6;      /* Main accent color */
    --secondary-color: #10b981;     /* Secondary accent */
    --bg-dark: #0f172a;            /* Dark background */
    /* ... more variables */
}
```

### 4. Update Social Links

Replace placeholder URLs in `index.html`:

```html
<a href="https://linkedin.com/in/your-profile" target="_blank">
<a href="https://github.com/your-username" target="_blank">
```

## 🌐 Deployment to GitHub Pages

### Method 1: GitHub Web Interface

1. **Create a GitHub Repository**
   - Go to [GitHub](https://github.com)
   - Click "New Repository"
   - Name it: `your-username.github.io` (for user site) or `Abhilash-Portfolio` (for project site)
   - Choose "Public"
   - Click "Create Repository"

2. **Upload Your Files**
   - Click "Upload files"
   - Drag and drop all portfolio files
   - Commit changes

3. **Enable GitHub Pages**
   - Go to repository **Settings** → **Pages**
   - Under "Source", select **main** branch
   - Click **Save**
   - Your site will be available at: `https://your-username.github.io/Abhilash-Portfolio`

### Method 2: Using Git (Command Line)

```bash
# Initialize Git repository
git init

# Add all files
git add .

# Commit changes
git commit -m "Initial portfolio commit"

# Add remote repository
git remote add origin https://github.com/your-username/Abhilash-Portfolio.git

# Push to GitHub
git branch -M main
git push -u origin main
```

Then enable GitHub Pages in repository settings as described above.

### Method 3: Using GitHub Desktop

1. Open GitHub Desktop
2. Click "Create New Repository"
3. Fill in the details and choose the folder location
4. Click "Publish Repository"
5. Enable GitHub Pages in repository settings
